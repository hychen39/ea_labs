Manage Items

Use Case

操作步驟:
  1. 在Home 頁面中按下 Manage Item 鈕進入 Manage Item Page進行清單管理頁面(Conversation 開始)。 Manage Item Page 一開始會顯示預設清單。
  2. 按下在項目旁的 Delete 鈕，移除該項目，並顯示更新後的清單。
  3. 按下 Refresh 鈕，在同一頁面顯示現有清單項目
  4. 按下 Home 鈕，回 Home 頁面，但清單內容仍保留目前現況
  5. 按下 End 鈕，結束清單內容的編輯，回Home 頁面。此時，重置清單內容為預設值。
