Insert an image using URL attribute

<h:graphicImage url="/resources/default/img/demo_img_01.jpg" />

Insert an image using library and name attributes (JSF2.0)

<h:graphicImage library="img" name="demo_img_02.jpg" />

Use library and resource name to access the image files

The default JSF ResourceHandler will always get the highest version to display if we don't specify the version.

Specify the resource version to display
<h:graphicImage library="defaut" name=" img/demoImg03.jpg/1_0.jpg" />

Not specify the resource version to display
<h:graphicImage library="defaut" name=" img/demoImg03.jpg" />

Insert an image using Resource EL-Resolver

Use the EL implicit object resource to access the resource:
<h:graphicImage value="\#{resource[' default:img/demo_img_03.jpg ']}" />
